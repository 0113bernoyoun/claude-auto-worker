# 🚀 Claude Auto Worker - Development Tasks

## 📋 프로젝트 개요
**프로젝트명**: claude-auto-worker  
**목표**: 차세대 Claude Code 자동화 도구 - 안전하고, 유연하고, 확장 가능한 워크플로우 엔진  
**기술 스택**: NestJS, Next.js, TypeScript, Node.js  
**개발 기간**: 16주 (2024년 8월 ~ 12월)

---

## 🎯 빠른 프로토타입 목표
**6주 내에 기본적인 워크플로우 실행이 가능한 프로토타입 완성**
- 실제 YAML 워크플로우 파일을 실행하여 Claude API 호출 및 Git 연동이 동작
- 사용자 피드백 수집을 통한 다음 단계 개발 방향 결정
- MVP 개발을 위한 핵심 기능 우선 구현
